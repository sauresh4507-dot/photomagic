# PhotoMagic - AI-Powered Background Remover

PhotoMagic is a premium web application that uses artificial intelligence to remove backgrounds from images with professional precision. Built with React, Vite, and Node.js.

## ✨ Features
- **Premium UI**: Sleek, glassmorphic design for a modern look.
- **AI Background Removal**: Powered by the remove.bg API.
- **High Definition**: Download processed images in HD PNG format.
- **Fast & Responsive**: Optimized with Vite for a smooth user experience.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- An API key from [remove.bg](https://www.remove.bg/api)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sauresh4507-dot/photomagic.git
   cd photomagic
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure the API Key:
   Open `server.js` and replace the `X-Api-Key` with your own key.

### Running the Application

Start the backend server:
```bash
npm run server
```

Start the frontend development server:
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Lucide Icons
- **Backend**: Node.js, Express, Multer, Axios
- **Styling**: Vanilla CSS (Custom Glassmorphism)
